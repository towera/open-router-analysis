# OpenRouter Model Usage Analysis

## Assignment

This repository contains an analysis of OpenRouter API usage data to understand global trends in large language model (LLM) adoption.

The project uses OpenRouter usage data to construct country-level indicators of model activity, including overall API token usage and the relative adoption of open-source models and models developed by Chinese organisations. These indicators provide insight into the evolving global landscape of AI model usage.

## Data

The following dataset is used in this analysis:

**OpenRouter API Usage Data** — Aggregated API usage logs from the OpenRouter platform capturing monthly token consumption by model and country.

The dataset includes information on:
- API token usage by country
- Model identifiers
- Monthly activity levels
- Model provider classifications

Additional processing steps classify models by ecosystem characteristics, including open-source status and Chinese model origin. Further details on data processing and methodology are documented within the notebooks and analysis pipeline.
